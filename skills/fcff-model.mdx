## What it does

The core DCF engine. Takes all prior outputs and runs the full FCFF valuation, producing an estimated value per share.

## The three stages

### Stage 1: 10-year FCFF projection

For each year 1–10:

| Line item | Computation |
|-----------|-------------|
| Revenue | Prior year × (1 + growth rate) |
| Operating margin | Linear convergence from current to target |
| EBIT | Revenue × operating margin |
| After-tax EBIT | EBIT × (1 − marginal tax rate) |
| Reinvestment | Δ Revenue / sales-to-capital ratio |
| **FCFF** | **After-tax EBIT − reinvestment** |

Growth rates decline linearly from the initial high-growth rate to the stable growth rate over the 10-year period.

### Stage 2: Terminal value

```
TV = FCFF(11) / (WACC(stable) − g(stable))
```

Where:
- **FCFF(11)** = year 11 free cash flow (at stable growth)
- **WACC(stable)** = cost of capital at maturity (may differ from high-growth WACC)
- **g(stable)** = stable growth rate (typically ≤ risk-free rate)

### Stage 3: Equity bridge

```
  PV of year 1–10 FCFFs
+ PV of terminal value
= Operating value of the firm
− Debt
+ Cash
− Employee stock options value
× (1 − probability of failure)
÷ Shares outstanding
= Estimated value per share
```

## Python engine

The math is implemented in `lib/dcf_engine.py` — a pure Python module with no AI dependencies. Every intermediate value is logged to the run transcript for auditability.
