# Value any company in a conversation

Valuation101 is a Claude plugin that runs Damodaran's FCFF valuation model through natural conversation. It fetches SEC data, computes WACC, projects free cash flows, and returns an intrinsic value per share — all while explaining every step.

## How it works

<Steps>
  <Step title="Pick a company">
    Say "Value Apple" or "What's Tesla worth?" — the plugin identifies the company and fetches live SEC EDGAR data.
  </Step>
  <Step title="AI builds the model">
    16 skills run in sequence: pulling XBRL data, computing trailing-twelve-month financials, estimating WACC, projecting growth, and running the DCF.
  </Step>
  <Step title="Get your intrinsic value">
    You receive a value per share with a full breakdown — revenue projections, margin convergence, reinvestment rates, and terminal value. Every number is auditable.
  </Step>
</Steps>

## Built on Damodaran's methodology

Every formula in this plugin maps directly to a cell in Aswath Damodaran's
[fcffsimpleginzu spreadsheet](https://pages.stern.nyu.edu/~adamodar/). The math is implemented in a tested Python library, so calculations are deterministic and auditable — the AI handles the conversation, not the arithmetic.
